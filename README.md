# sql-sales-data-analysis-project
SQL project analyzing sales data to generate business insights such as total sales, customer behavior, product performance, and regional sales distribution using SQL queries, aggregations, and joins.

  SQL Sales Data Analysis Project
📌 Project Overview

This project focuses on analyzing sales data using Structured Query Language (SQL) to extract meaningful business insights. The analysis is performed on a relational database consisting of multiple tables that store information about sales transactions, customers, products, and dates.

The main goal of this project is to demonstrate how SQL can be used to analyze business data, identify sales trends, evaluate product performance, and understand customer behavior.

This project is suitable for beginners learning SQL and for showcasing SQL skills in a data analytics portfolio.

🎯 Project Objectives

Analyze sales data to generate meaningful insights.

Calculate total sales and total quantity sold.

Identify top-performing products and categories.

Analyze customer purchase behavior.

Evaluate regional sales performance.

Demonstrate the use of SQL joins, aggregations, and filtering.

🗂️ Dataset Description

The project uses four main tables:

1. Sales Data (sales_data)

Contains sales transaction information such as:

Order ID

Product ID

Customer ID

Quantity

Sales Amount

Order Date

2. Product Data (product_data)

Contains product-related information:

Product ID

Product Name

Category

Price

3. Customer Data (customer_data)

Stores customer details:

Customer ID

Customer Name

Region

4. Date Table (date_table)

Contains date-related information used for time-based analysis.


📊 Key Analysis Performed

Some of the key analysis tasks include:

Viewing and exploring sales data

Finding total sales and total quantity sold

Calculating sales by customer

Identifying top 3 highest sales orders

Analyzing revenue by product category

Finding the highest sales order

Calculating sales by region

Counting customers by region

Analyzing yearly sales trends
